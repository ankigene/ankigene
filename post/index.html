カードの内容をブラウザーの中で検索するアドオン

AnkiWeb に公開された「Search browser for selected words」というアドオンは、Anki のカード面で選択した内容を検索して、カードブラウザー内に結果を表示する機能を提供します。別のカードに同じ項目が含まれていないか調べるのに役立ちます。

米国医師国家試験のための Anki の活用法

医師アメリカ留学のためのブログ～USMLE勉強法、MPH～ の投稿「USMLE STEP1 勉強法　各論１ 」では、米国医師国家試験 (USMLE) に合格した著者の Anki 活用法を取り上げています。カードの内容や分量、学習ペースなどを具体的に紹介しています。医学生の Anki の活用法は英文では簡単に見つかりますが、日本から医学留学した方が自らの体験を元に日本語で書いている点が大変貴重です。

AwesomeTTS による読み上げ設定チュートリアル

萌え萌えmoebuntu の投稿「語学学習アプリ「Anki」の「音声読み上げ設定」　２、3 」は、二回にわたる AwesomeTTS のチュートリアルです。英語のテキストを読み上げるの方法から、TTSで読み上げる音声ファイルを一括作成する手順まで、画像をふんだんに使って丁寧に説明しています。PCとスマホで同じTTSの音声を聞きたい方にはお勧めの内容です。

ペーストする画像の大きさを変更するアドオン

AnkiWeb に登録された “Image Resizer – Resize the image when pasting” は、フィールドに画像を貼り付ける時に指定の大きさを変更するアドオンです。

メディアの登録を強化する JSON 読み込みアドオン

AnkiWeb で公開のアドオン “Json Importer” は、Anki の読み込み機能に JSON 形式のサポートを追加します。読み込みデータのフィールドの内容を画像や動画のパスにすると、自動的にメディアファイルとして取り込む機能を提供しています。
