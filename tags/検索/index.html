カードの内容をブラウザーの中で検索するアドオン

AnkiWeb に公開された「Search browser for selected words」というアドオンは、Anki のカード面で選択した内容を検索して、カードブラウザー内に結果を表示する機能を提供します。別のカードに同じ項目が含まれていないか調べるのに役立ちます。

単語単位でフィールド検索をするためのアドオン

Anki アドオン “Word search and wildcard field name” は、カードブラウザーでのフィールド検索にワイルドカードの使用や単語単位での検索の機能を追加します。
